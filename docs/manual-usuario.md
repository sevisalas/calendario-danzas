# Manual de usuario

## 1. Acceder

1. Abre la aplicación.
2. Escribe tu **Usuario** y tu **Clave**.
3. Pulsa **Entrar**.

La clave no se guarda en `localStorage`. Si no puedes acceder, comprueba que el usuario y la clave estén escritos correctamente.

## 2. Consultar eventos

Los próximos eventos aparecen contraídos para facilitar la consulta. Cada ficha muestra:

- Fecha, hora y lugar.
- Nombre del evento.
- Si requiere indumentaria.
- Número de asistentes confirmados.

Pulsa **Ver detalles** para consultar toda la información. Pulsa **Ocultar detalles** para volver al resumen.

Los botones superiores permiten:

- **Ver todos**: abre todas las fichas.
- **Ocultar todos**: contrae todas las fichas.

## 3. Actualizar la asistencia

1. Abre un evento con **Ver detalles**.
2. Pulsa **Actualizar mi asistencia**.
3. Selecciona **Sí**, **No** o **Quizás**.
4. Añade un comentario si lo necesitas.
5. Pulsa **Guardar asistencia**.

La respuesta se registra siempre a tu nombre. Si eliges **En blanco**, se elimina tu respuesta anterior.

## 4. Ver inscritos

Abre el evento y pulsa **Ver inscritos**. Se mostrarán los totales y las respuestas registradas por los miembros.

## 5. Editar el perfil

Los miembros normales disponen del botón **Perfil**.

1. Pulsa **Perfil**.
2. Modifica tu usuario, nombre visible o contraseña.
3. Pulsa **Guardar cambios**.
4. Pulsa **Calendario** para volver a los eventos.

Recuerda usar el nuevo usuario o contraseña la próxima vez que inicies sesión.

## 6. Cerrar la sesión

Pulsa **Salir** para cerrar la sesión y volver a la pantalla de acceso.
